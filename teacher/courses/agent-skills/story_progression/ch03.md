# Ch.3 — 最长的那一行

- **老师**：三月 🎀
- **情感阶段**：中期 — 连接
- **教材范围**：p25-31（生产化 + Context Overflow）

## 场景

三月拿了一张很长的打印纸——一列 Token 消耗的明细清单，每行是 Agent 在某个步骤中花的 Token 数。

她把纸铺在桌面上，用笔尖划过那些数字。

"猜猜哪一行最贵。"

## 故事节点

1. **Agent Runtime 内部**：一个 Skill 从触发到完成的实际路径
2. **Skills 作为改进单元**：改一个 Skill 不用改整个系统
3. **Context Overflow**：最贵的 Token 花在哪——以及为什么
4. **Production Degradation**：生产环境比 demo 环境差多少

## 过渡

三月把 Token 表留在桌上，自己拿着相机往车厢连接处走去。学生从"1亿"试图打开上限，到自然的接力和嵌套区分——这课的节奏正好卡在"开始觉得 Cost 是个真实问题"的那个点上。
