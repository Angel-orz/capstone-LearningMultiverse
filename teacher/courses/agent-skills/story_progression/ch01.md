# Ch.1 — 从"已知"出发

- **老师**：三月 🎀
- **情感阶段**：初期 — 信号
- **教材范围**：p6-11（Skill 解剖 + 两条路径）

## 场景

你上车的时候，三月面前摊着几份打印出来的文档——不是教材，是从某个项目里打出来的 SKILL.md 示例。

她没抬头——指了一下对面。"帕姆已经在了。"

帕姆坐在你旁边，尾巴在座椅边缘轻轻摆了一下。桌面上没有小刷子——她今天不是以列车长的身份来的。

三月抬起头。"好——今天有一个基础问题要先确认：在你心目中，一个 Skill 长什么样？"

## 故事节点

1. **Skill 解剖**：目录结构（SKILL.md + scripts/ + references/ + assets/）
2. **Progressive Disclosure**：元数据总是在上下文中 + 详细内容按需加载
3. **两条路径**：翻译已知（Translating Known Workflows）vs 结晶新知（Crystallizing New Capabilities）
4. **边界区分**：Skill vs MCP vs AGENTS.md——三层分工的实操案例
5. **Skills 作为程序性记忆**：理论背景

## 过渡

三月没擦白板。帕姆走之前用刷子扫了扫黑板下缘——只扫了灰尘，一个字没擦。
