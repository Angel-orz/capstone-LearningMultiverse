# Ch.4 — 技能的技能

- **老师**：丹恒 🗡️
- **情感阶段**：中期 — 连接
- **教材范围**：p32-38（Meta-Skills + 组合）

## 场景

丹恒在白板上画了一个图——一个 Skill 箭头指向另一个 Skill。

那个指向关系上方有一行小字备注："我写的。"

他转过身来。"教材上管这叫做 Meta-Skills——一个能生成或修改其他 Skill 的 Skill。"

他停了一下，让大家理解这个自我指涉。

"这听起来像递归——实际上更像是'教一个人怎么教另一个人'。"

## 故事节点

1. **Meta-Skills 机制**：Skill 生成 SKILL.md
2. **Self-Improving Skills**：运行 → 评估 → 修改自身
3. **局限**：什么时候这个循环会崩
4. **DAG Orchestration**：多个 Skill 的有向无环图执行
5. **Capability Profiles**：环境打包 + 能力描述
6. **Context Debt**：当 Skills 积累到一定程度后的维护成本

## 过渡

丹恒关掉白板前看了一眼四张图。学生从 DAG 到 Convergence Problem 到 Context Debt 全部覆盖，课后总结五层无遗漏。"没有需要纠正的"——丹恒最高级别的认可。
