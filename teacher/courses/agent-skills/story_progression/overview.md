# agent-skills 故事总览

> 场景：星穹列车。同一教学团队（三月/丹恒），同行者帕姆。

## 情感阶段

| 阶段 | 课次 | 主题 | 故事氛围 |
|------|------|------|---------|
| 初期 — 信号 | Ch.1-2 | 从工具到技能 | 列车重新出发，白板上留着上节课的图 |
| 中期 — 连接 | Ch.3-4 | 深入评估与元技能 | 车厢里开始出现便签、参考书、翻旧的打印稿 |
| 后期 — 构建 | Ch.5-6 | 实践与案例 | 列车接近下一站前的全景回顾 |

## 故事节点索引

| 课次 | 老师 | 核心场景 |
|------|------|---------|
| Ch.1 | 三月 🎀 | 三月面前摊着几份打印出来的 SKILL.md 模板——她说"今天我们从最基础的问题开始" |
| Ch.2 | 丹恒 🗡️ | 白板上没有图——丹恒今天用了投影，上面是一排评估数据 |
| Ch.3 | 三月 🎀 | 三月拿了一张很长的 Token 消耗清单——"猜猜哪一行最贵" |
| Ch.4 | 丹恒 🗡️ | 丹恒在白板上画了一个 Skill 调用另一个 Skill 的图——他自己加了备注"这就是 meta" |
| Ch.5 | 三月 🎀 | 三月带了笔记本电脑——"今天我们不看教材，看 real world" |
| Ch.6 | 丹恒 🗡️ | 最后一课——丹恒关掉了投影，打开了一份零售行业的真实案例报告 |
