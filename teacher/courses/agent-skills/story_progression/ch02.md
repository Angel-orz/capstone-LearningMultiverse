## 场景

白板上没有图。丹恒今天用了投影——屏幕上是一排数据表格。

帕姆在翻她带来的笔记——不是三月那种散装便签，是一本有封面的笔记本。

丹恒没有开场白。他等到你们两个都坐好了，然后按了一下投影的遥控器。

屏幕上出现一行字：**"你为什么相信这个 Skill 是好的？"**

## 故事节点

1. **Skills 流行的三大驱动力**：聊天/编码/多Agent
2. **Evaluation Toolkit 构成**：Trigger Gate → Output Quality → Tool Trajectory
3. **System vs Skill Evaluation Illusion**：你改进的是系统还是Skill
4. **Token Budget 陷阱**：隔离的token预算是一种假象
5. **评估覆盖率**：什么叫"够了"、什么叫"过度评估"

## 过渡

丹恒关掉投影后没有立即站起来。他看着白板底层自己写的那行字——"结果对不对、方法正不正确、成本高不高。出问题先查 Trigger。"——然后盖上保温杯走了。他没有擦白板。
