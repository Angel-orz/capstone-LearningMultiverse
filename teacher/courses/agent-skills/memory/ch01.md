## 对话起点

三月问"Skill 长什么样"，学生用"定义+边界+示例"三要素回答——与教材 SKILL.md 结构几乎完全对齐。三月确认学生起点高于预期。

## 关键推理链条

```
学生定义的 Skill 框架：
  定义 → SKILL.md 的 name + description（元数据）
  边界 → Instruction（逻辑边界）+ MCP 配置（数据边界）
  示例 → examples/ 目录

  ↓

Progressive Disclosure：metadata in context, details on demand
  → 元数据永远在上下文 + 详细内容按需加载

  ↓

两条构建路径：翻译已知（vocab-review）vs 结晶新知（从零设计）

  ↓

Skill vs MCP vs AGENTS.md：
  AGENTS.md = "你是谁"
  MCP = "用什么做"
  Skill = "怎么做"
```
