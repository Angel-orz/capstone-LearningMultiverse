## 对话起点

三月用"电饭煲→插线板→台灯"类比引入 MCP，但学生直接指出了类比中的逻辑断层（"为什么电饭煲插进去台灯亮了"）。三月当面承认了翻车，贴了一张便签在显示器上。

## 关键推理链条

```
MCP 三层协议（发现/配置/连接）
  → NxM 原型问题：N 个 Agent × M 个工具 = N×M 个接口
    → MCP 线性化解法：N + M
      → MCP 调试方法（Inspector/stdio/SSE）
        → Vibe Coder Toolkit 最佳实践
          → A2A 协议引入（Agent Card/Discovery/Task 生命周期）
```
