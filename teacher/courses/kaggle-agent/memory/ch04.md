## 关键推理链条

```
无依赖 Agent → 并行调用（学生选择）
  ↓
有依赖 Agent → 串联 Pipeline（学生选择）
  ↓
多个 Agent 复杂依赖 → Agent Team + Orchestrator
  ↓
边界问题 → Bounded vs Unbounded Domain
  ↓
GOTO Problem → 传证据不传结论
  ↓
A2A 三要素：Agent Card / Discovery / Task 生命周期
```
