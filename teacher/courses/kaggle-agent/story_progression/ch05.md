# Ch.5 — 暖房里的积木盒

- **老师**：三月 🎀
- **情感阶段**：后期 — 构建
- **暗线密度**：1（碎片 2 触发窗口 — Ch.5 课后）
- **教材范围**：Day 3 p1-35（Agent Skills 构建与评估）

## 场景

你到暖房的时候，三月还没到——但桌上已经摆好了东西。

不是教材，不是打印的论文。

是一盒积木。儿童玩具那种——大的、彩色的、边缘光滑的塑料积木。

旁边放着一张便签，上面是三月歪歪扭扭的字：

*"技能就像积木。每个积木做一件事——但组合起来可以做任何事。等我回来！我去泡茶！——三月（刚刚的字为什么要写得那么正经）"*

你听到走廊尽头茶水间的方向传来壶响——然后是她的脚步声，比水先到。

"诶！你看到积木了对吧！那我们就从这个开始——"

她坐下来，把一杯茶推到你面前，自己那杯冒着热气她没喝——她急着开始。

## 故事节点

1. **开场**：一盒积木，一个类比——"每个 Agent Skill 就是一块积木：有接口、有功能、可以和别的拼在一起"
2. **Skill 解剖**：Trigger（什么时候触发）→ Instruction（怎么做）→ Tools（用什么做）→ Examples（预期输入输出）
3. **两条构建路径**：翻译已知（把你日常会做的事写成 Skill） vs 结晶新知（从零创建新 Skill）
4. **边界判定**：Skill vs MCP vs AGENTS.md——什么情况用什么
5. **评估方法**：输出质量、工具轨迹、Token 预算——怎么看一个 Skill 好不好
