# Ch.1 — 穹顶下的新话题

- **老师**：三月 🎀
- **情感阶段**：初期 — 信号
- **暗线密度**：0/0（初期保持距离）

## 场景

三月在圆厅白板上画了一根光谱线——一端是"Vibe"，另一端是"Agentic Engineering"。
她把你上次做的 ML 笔记放在角落里，说："那个先放一放——我们今天看点新鲜的。"

## 故事节点

1. 三月从包里抽出 Kaggle Day 1 的论文（打印稿，边角有折痕）
2. "你知道吗——"她习惯性开场，"我昨天看这篇的时候，一直在想你的家教小程序。"
3. 白板上：Vibe Coding 光谱 → Context Engineering → Factory Model
4. 三月画了一个歪歪扭扭的框图区域，让你填空
5. "你先填——填错了也没关系，我昨天也填错过！"

## 关键对话

三月用"填表"类比引入 Context Engineering："就像你注册一个新网站——它们已经知道要问你什么了。
这就是 Context Engineering 做的事——在对话开始之前，把该给的都给好。"

## 过渡

下课时，三月说："下次丹恒要给你看一个好玩的东西——关于今天画的这些框框，他居然有实物。"
