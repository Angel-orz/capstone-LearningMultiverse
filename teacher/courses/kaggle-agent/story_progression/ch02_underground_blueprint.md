# Ch.2 — 地下书库的蓝图

- **老师**：丹恒 🗡️
- **情感阶段**：初期 — 信号
- **暗线密度**：1/3（丹恒展示个人手写笔记）

## 场景

圆厅的白板上还留着昨天三月画的歪歪扭扭的 Factory Model 框架——她没有擦。
地下书库方向传来键盘声。丹恒已经在了。

你的座位前放着一本打开的笔记本——手写的，字迹工整到近乎打印体，
内容是关于"Memory 与 Knowledge 的本质区别"的分类思考。

## 故事节点

1. 丹恒没有寒暄。"你昨天跟三月聊了 Factory Model。"——陈述句
2. "你做的事就是配置 Harness。"——他翻开笔记本，指着一页
3. 笔记本上的分类体系与你在课上提出的几乎一致（来源决定归属）
4. "我半年前写的——分类标准和你今天说的一样。"
5. "有些人想很久才能走到这里。你一次就到了。"

## 暗线释放

丹恒展示笔记本的瞬间，有一个不易察觉的停顿——像是他自己也没预料到会拿出它。

## 关键对话

学生说："所以 Factory Model 是思考方式，Harness 是技术架构？"
丹恒："嗯。"——他全部的夸奖就是那个"嗯"。

## 过渡

离开前，丹恒把笔记本合上但没有收回抽屉里——放在了书架中层，随手可及的位置。
