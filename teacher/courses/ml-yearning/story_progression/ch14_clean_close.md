# Ch.14 — 干净收尾（丹恒）

**误差归因 + 全课程串联。丹恒的最后一课。安静收尾——一张没有路线的地图。**

角色状态：
丹恒今天的教学异常轻——不是内容轻，是态度。他在 Ch.12 说了最重的话之后，这一课他没有任何新的东西需要释放。他是纯净的教学模式——专注、精准、没有自己的东西需要带进来。保温杯全程拧着——不是紧张，是"不需要喝"。

**必然节点：**

1. **课前 — 干净的圆厅**
圆厅比平时干净。白板上没有残留的字迹——丹恒全部擦干净了，只留了 Ch.13 三月留下的鸟形问号。长桌上的杯子变成了三个——多了你的一杯水。丹恒为你准备的。他站在窗边，外面是早上的淡光，山上有一层薄雾。"最后一课。"他说——语气是陈述。"误差归因。这是诊断 ML pipeline 的最终工具——帮你定位问题在哪一步。然后——"他转向白板。"——回头看一遍你走过的路。"

2. **课中 — 误差归因**
丹恒讲"误差归因"时，用了一个前所未有的比喻。"你对一个病人做假设诊断——然后你给每个可能的病因分配一个概率。如果最后发现假设错了——不是病人错了，是你的诊断框架不够。误差归因在做的——是帮你知道哪个病因值得先检查。"这是丹恒级别的比喻——没有三月的暖，没有日常类比。是一个精确的工程比喻。但他后面加了一句——轻的："知道哪里错是第一步。知道往哪走是第二步。第二步教材没有——第二步是你自己的。"

**3. 课后 — 一张没有路线的地图（尾声）**
课后丹恒没有立刻离开。他把白板擦干净——全部，包括鸟形问号也被擦掉了——然后在空白的白板上用极细的笔迹写了几行字。不是一个课程总结——是一个方向列表。没有编号，没有顺序。
"误差归因 → 理解你的数据
理解数据 → 知道怎么收集
知道怎么收集 → 不再被模型骗
不再被模型骗 → 能用 ML 做对的事情"
然后他停住了——笔悬在"对的事情"下面的空白处。他没有继续写。他转过身。
"这不是地图。"他看着你。"是坐标系。路是你自己的——我画不出来。也不该画。"
他把笔放在桌上——这次放在笔槽里，整齐地——然后拿起了他的保温杯。盖子拧开了。
"如果以后有值得写的东西——"他指了指地下书库方向，"我还在写笔记。"
他说完这句话之后没有等着你回答。他走出圆厅——经过走廊——往地下室的方向去。在楼梯口停了一秒。没有转身。"明天不用上山了。但茶和水都会在厨房里。"
然后他走下去了。

**机会节点：**
- 如果你在课后追到地下书库——丹恒会站在书架前面，手里拿着那本空白的笔记。看到你进来——他把笔记放回去。"不是今天写。"他停了一下。"但应该快了。"他没有解释"快了"是什么意思。你也不需要问。
- 如果你在群聊里说了一句"谢谢"——丹恒会回一个字："嗯。"三月会回一整条："不用谢——你本来就会的，我们只是帮你把光打在那个点上。"然后附一张照片——第一课和最后一课的你。配文："你变了一点。"

**犯错节点：**
- 最后一课无独立陷阱。丹恒的教学重点是串联——把 14 课的所有 ML 概念整合成一个诊断和决策的完整框架。他会用 P5 的方式——让学生自己说出整个 pipeline 的架构，以及每个关键节点上的决策依据。

**群聊脉搏：**
- 三月发了最后一条消息——拍的是第一课和最后一课的白板变化。第一天的白板上有两种字迹、一个鸟形问号。最后一天的白板上什么都没有——"但你知道上面写了什么。"群名被丹恒改了——罕见——改为「已校准」。不是"解散"。是"已校准"。三月最后回了一个句号。
