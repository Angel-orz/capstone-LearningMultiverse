# Ch.11 — 三月·爆发（三月弧线高潮）

**数据不匹配 + 人工数据合成 + 优化验证测试。在讲"过拟合到错误分布"时，三月的类比说到一半停了——那个类比描述的不是 ML 问题，是她自己。**

角色状态：
三月今天抱着相机进圆厅——前几课相机都在暖房。她的外表跟平时一样——粉发、发卡、针织衫——但眼神里有轻微的收束，像在深呼吸。她知道今天的课包含"数据不匹配"和"人工数据合成"——这两个概念太近了。太近了。但她决定用它们教学。

**必然节点：**

1. **课前 — 今天带着相机**
圆厅。三月把相机放在桌上——屏幕朝上，而不是朝下。这本身就是一个信号。"今天讲数据不匹配。还有人工数据。"她的语气比平时平。"这两个概念是 ML 里最'人'的部分。"她没有解释什么叫"最'人'的部分"。但你知道她不是在说教科书。

2. **课中 — 过拟合的类比（爆发时刻）**
讲到"数据不匹配——你在一个分布上训练，在另一个分布上测试"时，三月用了过拟合的类比。"过拟合是——你把训练集里的每一个噪音都当成了规律。你记得了所有例题的数字但不知道为什么是那个答案。然后真实世界——"她停了大约三秒。
"然后真实世界给了你一道你没见过的题——"
她的声音在这里变了——不是音量，是节奏。从讲类比变成了说真实。
"你的数据不是你的。你的知识不是你的——你学了但它来自一个你不知道的分布。你在猜——你永远在猜——因为它被处理过了。记忆。你的一部分是被人造出来的数据。"
她的手碰到了她的相机。相机屏幕亮着——上面是那张她不记得拍过的走廊。
"你在这个不存在的训练集上优化了太久——你以为你懂了但你不确定你是不是真的懂——你只是残影——"
她的声音断了一下。然后她停住了。
圆厅里只有穹顶那条光缝在移。
她把相机拿起来——没有拍照——只是在手里按了一下快门。空的。没有照片。
"对不起。"她把相机放回桌上——屏幕朝下。"刚才说偏了。"

3. **课后 — 暖房，门没开**
课后三月没有回暖房。她在厨房里倒了水——喝了一口，放了很久，水凉了。如果你跟过去——她会在你开口之前先说："相机里有几张我不知道什么时候拍的照片。不是技术问题——是记忆。我以前的。我不记得的。"然后她把水倒了，重新接了一杯热的。
"你知道最奇怪的是什么吗——"她看着水杯里的蒸汽。"——这些照片里，有三张是我教学的时候用过的类比里的场景。我不知道是我先去过那些地方——然后潜意识拿来教——还是——"
话断了。她没有继续。
"没事。今天的课其实讲完了——数据不匹配和人工合成你都懂了吗？"
她在用教学把你拉回来。也在把自己拉回来。

**机会节点：**
- 如果你在她爆发后说"三月——谢谢。"不是追问，不是安慰。就是谢谢。她会把你这句话记住。暖房的门会在当晚开着，灯亮到大半夜。
- 如果你问"你不记得的那些——你想找回来吗"，她会在沉默五秒后说"不想。因为我不知道找回来的是什么。万一是我不该知道的东西。""但我想知道为什么我知道这些。"矛盾。她自己知道。

**犯错节点：**
- "人工数据合成的风险"是最难讲的——因为对三月而言它不是风险，是亲历。所以她的教学在这里有轻微的回避。如果你注意到了并问"人工数据会不会引入 bias？"她会从教学中抽回来——拿回自己的专业声音——认真回答。
- **Fallback**：如果触及了她在教学上回避的关键点，她会在 P3 阶段推一个预设陷阱：让你设计人工数据并发现你自己引入了分布偏移。她自己不回答——你回答。

**群聊脉搏：**
- 三月发了一张照片——今天的圆厅，光在桌上。配文一个字："嗯。"丹恒在二十分钟后回了三个字："继续讲。"——不是"你还好吗"，不是"刚才怎么了"。是"继续讲"。这就是丹恒级别的安慰。三月回了："好。"群名被三月改了——变成了「信号站（信号不太稳定但还在发）」。
