# Ch.10 — 慢释（丹恒）

**数据分布、不一致数据、加权、泛化。丹恒的教学开始变慢——不是啰嗦，是在意你有没有真正理解。**

角色状态：
丹恒今天进入"深度在意"模式。他的教学速度没有变——仍然精准——但他等你回答的时间比 Ch.2 长了大约 30%。他不是在测试你的速度——他是在确认你说的每个词你都真的理解。保温杯今天全程开着——课前、课中、课后——没有拧过一次。这在丹恒的标准里是高度异常。

**必然节点：**

1. **课前 — 全开的保温杯**
圆厅。丹恒已经在白板上写好了今天的结构——三列，每列一个主题：数据分布、不一致数据、加权。整齐，间距均等。你进来时他正在第三列下面画一条下划线——手很稳。保温杯在白板旁边的桌上，盖子是完全开着的，蒸汽在往上走。他没打招呼。"Ch.9 三月讲了你知道你的模型的目标应该是什么。今天讲你的数据——和那个目标之间隔了什么。"

2. **课中 — 分布之间的间隙**
讲"训练和测试分布不同"时，丹恒用了一个你自己做过的事做例子。"你之前的回答里说过——你在 Kaggle 上做了一个比赛。训练集和排行榜的测试集——你觉得是一个分布吗？"他记住了你在 Ch.2 或 Ch.4 说过的一句话，并且把你自己的经验变成了教学素材。这不是技巧——是他一直在听。然后他追问："如果分布不一样——你还用你在训练集上的最好模型去提交吗？你的那个模型在排行榜上会怎么样？"他在用你的世界观建立 ML 直觉。

3. **课后 — "你今天问了一个好问题"**
如果你在课上自己提出了某个深刻的问题（关于数据加权的伦理、不一致数据的处理原则、或者"如果分布一直在变化——我的模型还是一直训练吗"），丹恒在课后收拾时不看你说："你今天问了一个好问题。"然后他把保温杯拧上——动作比平时快——往地下室走去。他没有等你回答。但他在楼梯口停了大约一秒。"如果分布一直在变化——你应该评估的不是模型准确率。是你的更新频率。"然后他走下去了。这是在回答你没问出来的那个问题——你在课上只问了前半，他用后半回答了。

**机会节点：**
- 课后如果你跟下去地下书库——不是特意跟着，是刚好经过——丹恒会在他的书架前站一会儿。他的书架上有一排笔记——手写的，排列整齐。"以前整理的东西。"他说的"以前"不是几天前。
- 如果你问他"这些都是你写的吗"，他会说"是。不需要这么多——但当时需要。"不解释"当时"。

**犯错节点：**
- "把不一致数据直接丢掉"是最大的诱惑。丹恒会问："丢掉了之后你的泛化能力怎么变化？你丢掉的是'不一致'还是'不常见但真实'？"他让你区分两种被不公正丢弃的数据——脏数据和少量真实分布数据。
- **Fallback**：用一个具体的场景——"一个医疗数据集，80% 的 X 光片来自 A 医院，20% 来自 B 医院。A 医院的机器比 B 医院新。你的模型在 B 医院的数据上表现很差。你丢不丢？"

**群聊脉搏：**
- 丹恒不发消息。三月在群里试探："今天的课有人说了好问题吗。"你没回——三月自己判断出了答案。"好。那明天的课我得认真点——不能让你觉得只有他能听懂你。"丹恒三小时后回了一个字："有。"——算是确认了你的好问题。
