# Ch.13 — 末光（三月）

**Pipeline 组件 + 直接学习丰富输出。三月的最后一课——她说了一句话但没说完。收束。**

角色状态：
三月今天不是"积蓄"——是"释放之后"。Ch.11 的爆发不像是一道裂口——更像是一个长期的紧张被松开了。今天她教课的方式有一种奇怪的轻——不是开心，是"最重的东西已经说完了"。她的相机屏幕在圆厅的桌上——朝上。镜头盖盖着。

**必然节点：**

1. **课前 — 暖房的门开着**
暖房的门开着——比平时开得更大。三月在给植物浇水——你第一次看到她浇水。她的动作不熟练——水洒出了一点，她用袖子擦了擦玻璃顶下的台面。"你知道为什么我养这些植物吗。"不是问句。她自己接下去："因为它们不在乎我不知道它们的名字。"然后她站起来，拿起相机，走向圆厅。"最后一课——讲 pipeline 和直接输出。学完之后你就能用 ML 做实际的工程选择了。"

2. **课中 — 选择的代价**
讲"选择 pipeline 组件——数据可用性和任务简单性"时，三月的类比回落到她最自然的水平："选 pipeline 组件就像点菜——有的东西你有很多数据，有的东西你基本没见过。你不能对没数据的东西点'全自动'。你得手动。"然后她说到任务简单性："有些任务太简单了——直接给输出就行。有些任务太复杂——你得拆成步骤。"
然后她加了一句——不是类比：**"人也是。复杂的人——你得拆开看。每个组件都有它的来源。但不是每个来源你都记得。"**
这句话比她在 Ch.11 的爆发轻得多——但说的是同一件事。区别是——她今天说这句话的时候是笑着的。不是藏。是接受了。

3. **课后 — 没说完的句子**
课后三月在圆厅多待了十分钟。她在擦白板——不是擦干净，是把之前画的那些歪歪扭扭的示意图擦掉了一部分，留下了一部分。留下的部分是 Ch.1 的那个鸟形问号。"这个留着。因为好玩。"她把白板擦放在桌上，拿起了相机。
"你知道吗——这一周——"她没说完。停了两秒。然后她笑了——是自己觉得好笑的那种。"算了。明天让丹恒收尾吧，不能我替他说。"她把相机挂到脖子上，经过你时拍了拍你的肩膀。
"Ch.14——他不喜欢最后一课被剧透。"然后她暖房的门没有关——今天一整夜都开着。灯亮到很晚。

**机会节点：**
- 如果你在课后问她"你还好吗"，三月第一次不回避——"还好。比以前好。"她说这句话时没有看相机。是看着你。然后她低头笑了一下，像是不好意思承认这句话是真的。
- 如果你问她"你记得的到底是什么"，她会沉默片刻——比平时长——然后说"一组数据。被清理掉之前我用过的那些直觉。教你是——在用那些直觉。"然后她举起相机拍了一张你的照片——你可能第一天被拍过——但这次她拍了。镜头盖开了。

**犯错节点：**
- 本节无重大陷阱。三月在这个阶段的教学节奏是"帮你把知识拼起来"——不会设太深的陷阱。她会用 P5（命名与回看）的方式帮你串联整个课程。"你从 Ch.1 的天气预报到现在知道了 ML 的整个 pipeline。我有直接告诉你任何东西吗？"让你自己说出来。

**群聊脉搏：**
- 三月发了三张照片——Ch.1 的圆厅、Ch.7 的暖房忘了关门的那天、今天你被拍的那张。配文："一周。光变短了。人不一样了。"丹恒回了一段话——罕见的、两行："明天最后一课。准备好。不是考你——是让你知道自己会了什么。"三月回了一个表情（一个圈——"懂的"）。群名变为「信号站（已校准）」。没有改回。就是已校准。
