# Ch.4 — 审视（丹恒）

**丹恒的第二课。他在判断你的动机——如果你通过了，他的态度会微妙变化。**

角色状态：
丹恒今天比平时更安静——不是心情不好，是在收集数据。Ch.2 中他看到了你的学习方式。现在他需要判断：你学 ML 是认真的，还是来打卡的。他的保温杯盖子拧得比平时紧。

**必然节点：**

1. **课前 — 教材里的"不完整"**
圆厅的白板上还留着三月昨天写的两家餐馆的数据。丹恒没有擦——直接在旁边加了两个公式。"误差分析。"他说，没有打招呼。"不是看准确率——是看你错在哪。"然后他停了一下——不是让你消化，是他自己看到了教材里的某一行。他用笔在教材的一个表述旁边打了一个很小的问号。"书上这里不够完整。实际标注错误有两种——标注员的主观错误，和数据本身的歧义。分不清这两个的话，你在改的是错的东西。"

2. **课中 — "你想用 ML 做什么"（条件触发节点）**
在讲误差分析的过程中——在你卡住、挣扎、然后自己理清了一个概念之后——丹恒问了一个不是教学的问题。语气和问"偏差的定义是什么"一样平。"你学这个——想用它做什么。"不是疑问句。是陈述句加一个句号。他在等。如果你的回答是真实的（不是"好找工作"或"热门"），他会沉默片刻——大约三秒——然后说"可以"。然后进入下一个知识点。"可以"这个评价你第一次听到。如果你回答了任何敷衍的东西，他不会说什么。但保温杯的盖子没有拧开。
（如果 Ch.4 没触发这个节点，它会在 Ch.6 再次出现。）

3. **课后 — 多了一句**
如果你在 Ch.4 的"动机问题"中给出了真实回答，丹恒在课后收拾时多说了一句话——背对着你，在擦白板。"明天同一时间。"这句话本身很普通。但在丹恒这里——他从不说社交套话。他说这句话意味着他决定认真教你了。

**机会节点：**
- 如果你在他问"你想用 ML 做什么"时反问"你呢？"——他会停住。白板擦停在手里。然后继续擦。"以前用来做一些事。现在——用来确保那些事不重来。"不多解释。但语气不冷——是沉的。

**犯错节点：**
- 如果你在标注错误分析（Ch.16）上急于下结论——"这些标签就是错的，修掉就行"——丹恒会问"你怎么确定是标签错了而不是模型错了？"如果一个标签被两个标注员给了不同的分类——谁的错？追到你发现"正确标签本身可能是模糊的"。
- **Fallback**：如果你发现不了，他会举一个具体例子——"一张照片，一只猫半身在阴影里。一个标注员标了'猫'。一个标了'不是猫'。谁对？"然后让你自己给出标准。

**群聊脉搏：**
- 三月在群里问："今天丹恒有没有说'可以'？" 你回复了。三月回了一个表情（极少量 emoji 之一——一个句号）。"他说'可以'——你知道这意味着什么吧。"丹恒没回复。但他没有否认。
