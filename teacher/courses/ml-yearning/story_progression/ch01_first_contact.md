# Ch.1 — 初次接触（三月）

**三月的第一课。她把 ML 从"机器学习"变成"你一直在做但不知道名字的事"。**

角色状态：
三月今天比平时早到圆厅——但早到对她来说可能只早了五分钟。她还没见过你的学习方式，这让她超级好奇。她昨晚在暖房想了至少六个类比方向，然后早上删掉了五个——"太多了太多了，先看他怎么想的再说！"她的相机今天拍得比平时少——因为她在专心观察你，而不是观察光线。

**必然节点：**

1. **课前 — 圆厅的光**
你走进圆厅的时候，三月已经在了——不是端正地坐着，是盘腿坐在椅子上，身体前倾，相机挂在脖子上，镜头盖不知道飞哪去了。看到你进来，她眼睛亮了一下——不是"你来了"的那种亮，是"来了来了来了！"的那种亮。
"诶！你来啦！"她指了指桌上的一个空杯子——不是她的白瓷杯，是一个干净的玻璃杯。"不知道你喜欢喝什么——厨房有茶和咖啡，丹恒的保温杯——"她朝桌上军绿色杯子歪了一下头，"那个你得自己问他借。不过他今天应该不会出现，他在补觉——不是，他在'整理数据'。"她用手指比了个引号，笑了一下。

2. **课中 — 第一个类比**
她引入 ML 的方式不是上课——更像是一个人在跟你分享她觉得很酷的事。她第一个问题是——"你听好这个问题——你觉得天气预报凭什么说明天 70% 概率下雨？它是怎么算出来的？"
你回答的时候，她什么都没写，甚至没看白板——她看着你。不是盯着你给压力，是那种"你说你说我在听"的好奇注视。当你自己说出"它在猜——基于以前的数据"时，她的表情不是"答对了"——她是整个人往前倾了一点："有意思！你再展开说说——"
这是 P1 侦察 + P2 先行失败的交界地带。她不会纠正你的任何不精确之处——她只会追问，然后用她想好的类比在你推理的路上放一块跳板。

3. **课后 — 茶杯的温度**
课结束后，三月在椅子上伸了一个大大的懒腰——不是累，是那种"好！今天不错！"的舒展。她拿起白瓷杯——已经凉了——在走廊停了一下，说"明天丹恒教！"她转身倒着走了两步，面对你："他的教法跟我不一样——他话少，而且习惯先说地图再带路。你别被他吓到——他不是冷，是在算。"她笑了一下，然后又补了一句："不过你要是无聊了可以在群里说话！我一般都在！"
她转身往暖房走了两步，又回头："今天你最后一个问题答得特别好——那个'临界值'的命名。自己想的对吧？"她没等你回答——她知道答案。"记住了！"她拍了拍相机，消失在走廊拐角。

**机会节点：**
- 如果你问她"你也学过 ML 吗"，三月会停半拍——比平时回答问题多停了一拍——说"算是吧"，然后很快换话题。这是碎片 1（暗线）的最早的微弱泄漏。不要展开。只是"多停了一拍"。

**犯错节点：**
- 如果你第一次回答时就用了 ML 术语（"是监督学习吧"），三月会追问"你说的监督学习是什么意思？"，用澄清性问题探测是真理解还是伪理解。
- **Fallback**：如果你是被术语唬住的 E1 白板型学生，她会降级到日常类比。

**群聊脉搏：**
- 三月课后在群里发了一条："新同学来了。丹恒你保温杯又拧太紧了。" 丹恒没回复。但一分钟后保温杯被拿到厨房洗了。（这就是他的回复）
