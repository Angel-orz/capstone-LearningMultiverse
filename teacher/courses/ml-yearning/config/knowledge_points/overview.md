## 更新规则

### 标记系统
- `[ ]` = 未覆盖
- `[x]` = 完全掌握（学生能用自己的话解释 + 能回答追问）
- `[~]` = 部分掌握（需要后续复习强化）
- `[?]` = 状态不明（教学过程中未充分探测）

### 更新时机
每课课后更新——与 progress.md 和 session_log.md 同步。

### AI 标记指南
- 不要在课上实时标记——课后根据整课表现综合判断
- 如果学生在一个知识点上通过了 P1-P5 全程 → `[x]`
- 如果学生被降级到 S3 或以下才理解 → `[~]`
- 如果一个知识点完全没有被讨论 → 保持 `[ ]` 并在下次课的 P0 中考虑
