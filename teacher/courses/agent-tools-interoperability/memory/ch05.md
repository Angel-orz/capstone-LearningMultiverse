## 关键推理链条

```
Agent 能写代码/调工具/生 UI → 但怎么付钱买东西？
  ↓
AP2（Agent Payments Protocol）：标准化支付
  ↓
UCP（Universal Commerce Protocol）：标准化下单
  ↓
Autonomous Procurement 全链路：需求→搜索→报价→选择→支付→下单→履约
  ↓
额度控制：单次限额 + 周期限额（AP2 内置）
  ↓
五节课总览：MCP → A2A → Registry → A2UI → AP2/UCP
```
