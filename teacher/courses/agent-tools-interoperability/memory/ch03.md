## 关键推理链条

```
Agent Card：身份 → 能力 → 调用方式 → 权限
  ↓
Registry：Public（公开市场）vs Private（企业内部）
  ↓
手动注册 = 授权开关（自己决定谁发现你）
  ↓
通信六步：Registry → Agent Card → A2A Task → Auth → 生命周期 → 完成
  ↓
A2A Extensions：对外动态调整暴露能力
```
