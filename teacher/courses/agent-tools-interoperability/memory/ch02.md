## 对话起点

丹恒以保温杯占座开场（暗示今天他是老师）。从"一个 Agent 做所有事"出发，引出 Contextual Overload 的三个后果：Token 成本爆炸、注意力稀释、延迟增加。

## 关键推理链条

```
单体 Agent → 所有事塞进一个提示词
  ↓
Contextual Overload → Token/注意力/延迟 三重问题
  ↓
内部专业化（Internal Specialization）：同一系统内拆分子 Agent
  ↓
外部专业化（External Specialization）：跨团队 Agent 通过 A2A 协作
  ↓
Bounded vs Unbounded Domain → 订单 Agent / 库存 Agent 边界案例
  ↓
GOTO Problem → 解决方案：传证据不传结论
```

## 核心概念

| 概念 | 掌握程度 | 备注 |
|------|---------|------|
| Contextual Overload | 概念 ✓ / 应用 ✓ | 学生拆解了三重后果 |
| 内部 vs 外部专业化演进 | 概念 ✓ / 应用 ✓ | 架构演进逻辑 |
