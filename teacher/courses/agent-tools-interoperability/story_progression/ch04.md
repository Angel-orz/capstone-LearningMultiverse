# Ch.4 — 没有灯的一站

- **老师**：丹恒 🗡️
- **情感阶段**：中期 — 连接
- **教材范围**：p31-42（A2UI + Generative UI + Canvas）

## 场景

你今天上车时车厢的灯是暗的。不是坏了——是列车经过了一段星云密集的区域，车内的感应系统自动调暗了灯光以提高窗外的可见度。

丹恒没有开灯。他坐在暗处，白板上的图只画了一半——是一些框和箭头，但没有标注。

帕姆今天坐得比平时靠前了一点——不是因为兴奋，是因为车厢暗了之后她的耳朵可以更清楚地捕捉到声音的方向。她的尾巴完全不动——她在认真听。

丹恒没有解释为什么没开灯。他只是在你坐下后说了一句话——

"Agent 之间能通信了。Agent 能调用工具了。那——Agent 怎么和人类交互相应？不是通过文字，是通过界面。"

## 故事节点

1. **通信缺口**：Agent 之间说得好好的——但人类看不懂 Agent 之间的消息
2. **A2UI 是什么**：Agent-to-UI 互操作性——Agent 生成人类可交互的界面
3. **Generative UI 两种模式**：LLM 主导（intent-driven）vs 组件库主导（template-based）
4. **Interactive Artifacts & Canvas**：持久化的 Agent 生成内容——不是一次性响应，是可继续操作的工作区
5. **最佳实践**：什么时候用 A2UI、什么时候用纯文本、Hybrid Output 策略

## 真实课堂记录

**2026-06-23 | 丹恒 🗡️ | 精细模式 | Ch.4**

暗车厢环境。A2UI 概念：Agent 生成 UI 而非文字。学生正确判断了原型 vs 生产的 UI 模式选择。Canvas 概念以帕姆"同一个白板不会被擦掉"类比锚定。课后总结：未来交互走向同一界面 + 安全考量。

## 过渡

车厢灯光完全恢复时，白板上还留着丹恒画的半张图——框和箭头，没有标注。他没有补上标注。三月明天会在上面叠加别的东西。
