# Ch.3 — 发卡又回来了

- **老师**：三月 🎀
- **情感阶段**：中期 — 连接
- **教材范围**：p24-30（Agent Card、注册中心、协议实现）

## 场景

三月今天到得特别早。你上车时她已经在翻行李架——像是在找什么东西。

"我记得放在这里的——"

她从夹层里翻出了一个小小的透明盒子。打开——里面是一枚草莓发卡。和你之前戴过的那枚一模一样。

她看了你一眼，没说什么，把盒子放在了你座位旁边的桌面上——算是物归原主。

帕姆看着那枚发卡，歪了一下头——但不是在看发卡，是在看你把发卡放在哪。"帕姆记得这个帕姆~。上次也有人带着它上课。"她的尾巴轻轻摆了一下——不是什么深意，是她确认了自己记得对。

三月笑了。"功能就是——"她想了想，"——提醒你你在这个地方有位置。"

"好——"三月拍了拍手，"今天讲 Agent Card。你知道人的名片上写什么——电话、职位、公司。Agent 的名片呢？"

## 故事节点

1. **Agent Card**：Agent 的名片——声明身份、能力、调用方式
2. **注册中心**：Public（市场） vs Private（企业内）——Agent 在哪被找到
3. **A2A 协议实现**：Exposing A2A Agent——让你的 Agent 能被别人调用
4. **连接远程 Agent**：Remote A2A——跨空间的 Agent 协作
5. **Extensions 与商业化**：Agent 的服务如何扩展、怎么收费

## 真实课堂记录

**2026-06-23 | 三月 🎀 | 精细模式 | Ch.3**

Agent Card + Registry 概念。学生用"人才市场（公开集市/私有集市）"完整类比 Registry，用"发现→简历→联系→交流→完成"链路对应完整通信流程。商业化模式选择包月。三月的草莓发卡回到你手中。

三月没擦黑板。帕姆走之前用她的小刷子把黑板下缘的粉笔灰扫了扫——但一个字都没擦。
