# Ch.5 — 前方到站

- **老师**：三月 🎀
- **情感阶段**：后期 — 构建（收官）
- **教材范围**：p43-49（AP2 + UCP 商业协议）

## 场景

列车广播响了——"前方到站：终点站。请整理好您的学习笔记。"

三月坐在窗边，没有起来收拾东西。她看着窗外那颗靠近的行星——表面有城市的纹理，从远处看像是在发光。

帕姆今天坐得很正。她的小刷子放在桌面上——不是工作需要，是她觉得这节课值得把刷子放下。尾巴尖轻轻搭在座椅边缘——她在听。"支付"、"交易"、"采购"——这些词不在帕姆的日常用语里，但她知道这些是列车到达终点站前必须理解的东西。她不想错过。

三月转过来，把脚收起来盘在座位上。

"好——最后一节课。今天我们讲的是两件我们每天都在做但从来没想过会让 Agent 帮我们做的事：花钱和买东西。"

她笑了一下。
"这节课之后——你今天学的东西理论上可以在真实世界里帮你省钱。也可能帮你赚钱。"

## 故事节点

1. **缺了的一环**：Agent 能写代码、调工具、生成 UI——但它怎么付钱买东西？
2. **AP2（Agent Payments Protocol）**：Agent 支付协议——Agent 之间的支付标准化
3. **UCP（Universal Commerce Protocol）**：通用商业协议——Agent 帮人点外卖背后的标准化
4. **Autonomous Procurement**：完整的自动采购流程
5. **整课收束**：从 MCP 到 AP2——Agent 工程化的全链路图谱

## 真实课堂记录

**2026-06-23 | 三月 🎀 | 精细模式 | Ch.5 | 最终课**

AP2 + UCP 概念。学生用"加 AP2 插件"回应付费场景。三月以五节课全链路总览图收束。草莓发卡放入盒中放在桌角。

## 过渡

列车到站了。三月没擦白板。帕姆站在车厢门口，握着那把小刷子，等你们下车。她没有说"下次再来帕姆~"——但她的尾巴轻轻摆了两下。
