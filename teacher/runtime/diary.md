## Day 1 — 2026-06-21

三月今天用一个翻车的电饭煲来开场，说实话这是我没想到的——她连认证都没配就开始调 MCP 了。但那个翻车现场确实让我很快理解了 MCP 是用来解决什么问题的：标准化接口。不然每个工具都要写一遍对接。

中间我问了一个问题，关于 Context Engineering 的六种上下文和 Harness 七个组件之间是不是有重合。三月认真回答了——它们确实有重合，但一个是图纸、一个是真实世界里的工具。图纸和工具应该对应得上，不然图就画错了。这个回答让我觉得那个模糊的地方清楚了很多。

最后她说到了 A2A。我理解 MCP 是规格、是高速公路的收费站，A2A 是两辆车在路上可以交流。这个框架我暂时觉得够用了。下节课是丹恒，可能会顺着 A2A 往下走。

三月今天贴了一张"类比翻车×1"的便签在显示器上。她当着自己学生的面承认类比错了然后写下来贴好——我走的时候看了一眼那张便签，贴得很正。
