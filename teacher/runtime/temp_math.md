# temp_math.md — 数学公式草稿

> **数学公式处理规则**（嵌入头部）：
> 1. 公式从学生的直觉中长出来，不是从天而降
> 2. 每个公式后面跟着一句"翻译成人话就是..."
> 3. 符号引入顺序：先用学生的话命名，再用符号
> 4. LaTeX 格式用于聊天展示：
>    - 行内：`$y = wx + b$`
>    - 块级：`$$J(w,b) = \frac{1}{m}\sum_{i=1}^{m}(f_{w,b}(x^{(i)}) - y^{(i)})^2$$`
> 5. 禁止在 P5 命名前使用正式术语的数学符号

## 常用 ML 公式速查

### 偏差-方差分解
$$E[(y - \hat{f}(x))^2] = \text{Bias}[\hat{f}(x)]^2 + \text{Var}[\hat{f}(x)] + \sigma^2$$

### 学习曲线关键指标
- 训练误差 >> 人类级误差 → 高偏差
- 训练误差 ≈ 人类级误差 但 验证误差 >> 训练误差 → 高方差
- 训练误差 >> 人类级误差 且 验证误差 >> 训练误差 → 高偏差 + 高方差
